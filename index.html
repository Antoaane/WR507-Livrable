Escape the H103

Que se passe-t-il ?! 😨

Vous vous retrouvez enfermé dans la salle H103, les portes sont fermées et un cadenas bloque la sortie principale…

Vous allez profiter de ce temps enfermé pour avancer sur vos projets, mais vous ne trouvez pas votre clé USB non plus. Elle était pourtant dans votre poche il y a quelques minutes.

Pas de panique. 😮‍💨

  • Il fait sombre, avez vous essayé d’allumer la lumière ?
  • Cette clé USB ne doit pas être bien loin, cherchez la dans la salle !
  • Branchez la à votre ordinateur portable pour commencer à travailler.
  • Les écrans se sont allumés, mais ce ne sont pas vos projets, vous n’avez pas l’impression que des informations supplémentaires sont disponible sur les écrans ?
  • A l’aide de vos connaissances sur les couleurs acquises en h009, complétez le code en reliant avec les correspondances.
  • Le cadenas à disparu, la porte est peut être ouverte ?
  • Bravo ! Vous êtes enfin sortie, mais vos projets n’ont toujours pas avancés…
